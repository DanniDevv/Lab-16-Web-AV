  # BACKEND

  ## Iniciar Proyecto
  ```
    npm install -g pnpm
  ```
  ```
    pnpm init
  ```
  ## Instalar Dependencias
  ```
    pnpm install express mongoose cors body-parser jsonwebtoken
  ```  
  ```
    install bcrypt
  ```

  # FRONTEND

  ## Instalar Dependencias
  ```
    npm install
  ```
  ## Iniciar Proyecto

  ```
    ng serve
  ```

> [!NOTE]
> URL Frontend = [localhost](http://localhost:4200/register)
